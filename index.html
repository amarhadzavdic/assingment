Prvi zadatak na ITAcademy

Naš prvi assingment na ITAcademy jeste da kreiramo i objavimo prvi website. Za uspješno izvršavanje assingmenta potrebne su nam sljedeće operacije:

  • Kreiranje HTML dokumenta
  • Instalacija Git-a
  • Kreiranje lokalnog Git repozitorijuma
  • Smještanje HTML dokumenta u Git repozitorijum
  • Kreiranje naloga na GitHubu
  • Kreiranje javnog repozitorijuma na GitHubu
  • Objavljivanje HTML dokumenta korištenjem GitHub Pages servisa
